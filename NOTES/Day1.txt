// TOPIC 1 //
📂 MONOREPO
DEFINITION

MONOREPO KA MATLAB HOTA HAI EK HI REPOSITORY KE ANDAR MULTIPLE PROJECTS AUR SHARED PACKAGES KO MANAGE KARNA.

EXAMPLE
CODEVERSE
│
├── APPS
│   ├── ADMIN
│   └── WEB
│
├── PACKAGES
│   ├── UI
│   ├── UTILS
│   ├── CONFIG
│   └── TYPES
│
└── PACKAGE.JSON
WHY MONOREPO?

✅ EK HI REPOSITORY MANAGE KARNI PADTI HAI.

✅ SHARED CODE DOBARA LIKHNA NAHI PADTA.

✅ SABHI PROJECTS KA VERSION EK SAATH MAINTAIN HOTA HAI.

✅ BUILD AUR DEVELOPMENT FAST HO JATA HAI.

// TOPIC 2 //
📄 PACKAGE.JSON
DEFINITION

PACKAGE.JSON NODE.JS PROJECT KI CONFIGURATION FILE HOTI HAI.

ISME PROJECT KI IMPORTANT SETTINGS AUR INFORMATION HOTI HAI.

ISME KYA HOTA HAI?

• PROJECT NAME

• VERSION

• SCRIPTS

• DEPENDENCIES

• DEVDEPENDENCIES

• WORKSPACES

HAMARA PACKAGE.JSON
{
  "name": "codeverse",
  "private": true,
  "version": "1.0.0",
  "workspaces": [
    "apps/*",
    "packages/*"
  ],
  "scripts": {
    "dev": "turbo run dev",
    "build": "turbo run build"
  },
  "devDependencies": {
    "turbo": "^2.9.18"
  }
}
PRIVATE : TRUE
MEANING

PROJECT KO NPM REGISTRY PAR PUBLISH NAHI KIYA JA SAKTA.

WHY?

✅ MONOREPO PROJECTS GENERALLY PRIVATE HOTE HAIN.

WORKSPACES
"workspaces": [
  "apps/*",
  "packages/*"
]
MEANING

NPM KO BATATA HAI KI

• APPS KE ANDAR JITNE FOLDERS HAIN

• PACKAGES KE ANDAR JITNE FOLDERS HAIN

YE SAB ALAG-ALAG WORKSPACES HAIN.

BENEFITS

✅ ROOT SE HI SABHI DEPENDENCIES MANAGE HO JATI HAIN.

✅ HAR PROJECT KO ALAG SE INSTALL KARNE KI ZARURAT NAHI HOTI.

// TOPIC 3 //
⚡ TURBO
DEFINITION

TURBO EK HIGH-PERFORMANCE BUILD SYSTEM HAI JO MONOREPO KO FAST BANATA HAI.

INSTALLATION
npm install -D turbo
WHY "-D" ?

KYUNKI TURBO SIRF DEVELOPMENT KE TIME USE HOTA HAI.

PRODUCTION ME ISKI ZARURAT NAHI HOTI.

BENEFITS

✅ FAST BUILD

✅ CACHING

✅ PARALLEL EXECUTION

✅ BETTER PERFORMANCE

// TOPIC 4 //

BENEFIT

✅ EK HI COMMAND SE PURE MONOREPO KO START KAR SAKTE HAIN.

// TOPIC 7 //
🔒 PACKAGE-LOCK.JSON
DEFINITION

PACKAGE-LOCK.JSON DEPENDENCIES KE EXACT VERSION KO LOCK KARTA HAI.

WHY?

✅ HAR DEVELOPER KE SYSTEM ME SAME VERSION INSTALL HOTA HAI.

✅ PROJECT HAR MACHINE PAR SAME TARAH WORK KARTA HAI.
