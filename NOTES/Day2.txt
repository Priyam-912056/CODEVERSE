# CodeVerse Notes – Region Selector (React + TypeScript)

## Goal

Ek professional **Region Selector** banana jisme:

* User apna country select kare.
* API automatically user ka country detect kare.
* Agar selected country aur detected country same ho to access mile.
* Warna access deny ho.

---


# Step 7 – useDetectCountry Hook

Hook banaya.

Purpose:

* API call
* loading manage karna
* detectedCountry return karna

Flow:

```
useEffect

↓

fetch()

↓

response

↓

JSON

↓

find()

↓

setDetectedCountry()

↓

return
```

Return:

```ts
return {
    loading,
    detectedCountry
}
```
